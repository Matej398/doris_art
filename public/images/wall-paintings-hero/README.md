# Wall Paintings Hero Background Images

This folder contains transparent PNG images used as decorative background elements in the hero section of the wall paintings page.

## Image Requirements

- **Format**: PNG with transparency
- **Theme**: Wall painting elements, art motifs, decorative patterns related to wall paintings
- **Size**: Various sizes (the code will scale them appropriately)
- **Naming**: Use descriptive names like:
  - `decorative-1.png`
  - `decorative-2.png`
  - `decorative-3.png`
  - `decorative-4.png`
  - `decorative-5.png`

## Current Configuration

The page is configured to display 5 decorative images:
- 1 image at top-left
- 1 image at top-right
- 1 image at bottom-left
- 1 image at bottom-right
- 1 image at center-left

## Styling

Images are automatically styled with:
- Low opacity (10-20%) for subtle effect
- Slight rotation for visual interest
- Responsive sizing
- Positioned behind text content

If an image doesn't exist, it will be hidden automatically (no errors).
